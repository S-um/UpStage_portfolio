- 코드 (Golang) 를 통하여 Kubernetes 관리할 수 있는 기술
- Nvidia Containerd 를 통하여 Kubernetes 상의 컨테이너에서 GPU 를 물릴 수 있는 기술
- 다양한 오픈소스 (ex. Ory) 를 시스템에 적용할 수 있는 능력
- React 기술
- upstage 에 대한 관심과 열정
